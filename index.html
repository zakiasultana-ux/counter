What is CounterJS?

⚡

The Problem

You need a reusable, stateful counter widget that encapsulates its own markup, events, and logic — without coupling state to the DOM or scattering handlers across your codebase.

🎯

The Solution

The Counter class uses OOP to keep state, rendering, and event binding in one place. Pass a CSS selector and an optional starting value — the component renders itself.

🔒

Key Behaviours

The counter floor is always 0. When at zero, the decrement and reset buttons receive the .counter__btn--inactive class. Every reset() logs Reset Activated!!! to the console.

Getting Started

Place a container element in your HTML and give it an ID. Import the class and instantiate.

HTML container

<!-- Wrapper element in your HTML -->
<div
  id="my-counter"
  class="counter"
></div>

JS instantiation

// main.js (type="module")
import { Counter } from './counter.js';

// Counter(selector, initialCount, label)
const c = new Counter(
  '#my-counter',
  0,
  'My Counter'
);

Counter — Parameters

Parameter Type Default Description
selector string — CSS selector for the container element. Required.
initialCount number 0 Starting count value. Reset returns to this value.
label string '' Optional heading rendered inside the counter widget.

Counter — Public API

Method Returns Description
increment() void Increases count by 1 and syncs the UI.
decrement() void Decreases count by 1. Prevents values below 0.
reset() void Returns to initialCount. Logs "Reset Activated!!!" to console.
getValue() number Returns the current count without modifying state.

StepCounter

StepCounter extends Counter using extends and super(). It accepts a third step parameter and overrides increment() and decrement() — every other behaviour is inherited.

Instantiation

import { StepCounter } from './counter.js';

// StepCounter(selector, initialCount, step, label)
const s = new StepCounter(
  '#step-counter',
  0,
  5,
  'Step ×5'
);

Override pattern

class StepCounter extends Counter {
  constructor(sel, init, step, label) {
    super(sel, init, label);
    this.step = step;
  }

  increment() {           // override
    this.count += this.step;
    this.update();
  }

  decrement() {           // override
    if (this.count <= 0) return;
    this.count = Math.max(0,
      this.count - this.step);
    this.update();
  }
}

StepCounter — Additional Parameter

Parameter Type Default Description
step number 1 Amount added or subtracted on each increment/decrement action.
